1.This is a Springboot application which can be started by running the main class StartApplication -> Run StartApplication.main()
	
2.It has controller, services and repository modules to fetch data and the project involves usage of H2 and Thymeleaf dependencies.

3.I have added unit test cases which is in /test/java folder, code can be tested using the test cases as well. (mvn -Dtest=Testclass test)

4.http://localhost:8080 is the home url which on clicking Login navigates to list of accounts(http://localhost:8080/accounts)

5.On clicking any of the accounts, the page navigates to the list of transactions for that particular account.
